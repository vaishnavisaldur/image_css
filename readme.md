>>Project name: Displaying an image on a webpage.

>>Overview of the project:
    >Download or clone the repository.
    >Place your image file in the project folder and name it image.jpg.
    >Open the project folder.
    >Double-click index.html or open it using any web browser.
    >The webpage will load with the image, button, and styled layout.

>> Features:
    >Uses semantic HTML structure.
    >External CSS stylesheet linked to the HTML document.
    >Image displayed inside a container (div).
    >nteractive button included below the image.
    >Gradient background for enhanced appearance.
    >Responsive layout using viewport units (vh and vw).
    >Proper use of box-sizing: border-box.
    >No Flexbox used.
    >No overflow: hidden used.
    >No position: relative or position: absolute used.
    >Page layout designed to avoid unnecessary scrolling.

>>Technologies used:
    >HTML5
    >CSS3

>>How to run:
    >Download or clone the repository.
    >Open the project folder.
    >Double-click index.html or open it using any web browser.
    >The webpage will load with the image, button, and styled layout.
